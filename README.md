# Saikat Portfolio (NES.css Retro Edition)

Upgraded single-page portfolio with NES.css + custom CSS + vanilla JavaScript.

## Highlights

- Expanded skills matrix with badges, progress bars, and hover tooltips
- 11 project cards with detail modal (challenge + solution + impact)
- RPG-like sections: XP bar, quests, achievements, active mission system
- New sections: services, testimonials, tech stack visualization, tech radar, blog placeholder, currently learning, GitHub activity panel
- Dev terminal command box with basic commands
- Easter egg key sequence, visitor counter, and rotating "currently building" status
- Loading overlay, pixel cursor, smooth scrolling, sticky nav, active section highlighting
- Theme toggle with localStorage persistence
- Accessible modal focus trap + ESC close
- Contact form validation and simulated submission toast

## Stack

- HTML
- CSS
- JavaScript
- NES.css (CDN)

## Run

Open `index.html` directly in your browser.
